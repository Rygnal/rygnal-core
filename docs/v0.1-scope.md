Rygnal Core v0.1 includes a working **local MVP** with core security interception and governance capabilities.

| Component | Details | Status |
|-----------|---------|--------|
| **Interceptor** | Central control point for tool requests | ✅ Complete |
| **Policy Engine v1** | YAML-based policy rules with simple matching | ✅ Complete |

| **Audit Logger v1** | Cryptographically signed audit logs | ✅ Complete |

| **Approval Workflow v1** | Basic approval request/response flow | ✅ Complete |

| Advanced approval workflows | Basic workflow sufficient | v1.0 |

## Migration Path

**Using v0.1 locally?** Here's how you'll migrate to v1.0:

1. **Policies** - YAML policies will continue to work. New OPA/Rego support is optional.
2. **API** - Core interceptor API will be backward compatible.
3. **Deployment** - Local deployment continues to work. New cloud options available.
4. **Tool types** - New tool types will be added, existing ones remain.

**No breaking changes planned between v0.1 and v1.0** (though API may expand).
